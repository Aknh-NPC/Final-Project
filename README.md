# 17 SDG — Act Now

A simple, accessible static website introducing the 17 United Nations Sustainable Development Goals (SDGs) with links to the official UN pages and suggested local actions.

Preview
- Open `index.html` in a browser. The site is responsive and uses a small CSS file at `css/styles.css`.

How to run locally
1. Clone the repository:
   git clone https://github.com/Aknh-NPC/Final-Project.git
2. Open `index.html` in your browser, or serve with a simple static server:
   - Python 3: `python -m http.server` then visit http://localhost:8000

What’s included
- `index.html` — the main static page with the 17 goals
- `css/styles.css` — basic styling and responsive layout
- `LICENSE` — license for the project

Contributing
